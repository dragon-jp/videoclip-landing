Features

01 / Organize

フォルダで、意味ごとに分ける

テーマ、気分、プロジェクト。並べ替えも入れ子も自由。動画の山を、読み返せる棚に。

02 / Think

メモで、見た理由を残す

なぜ保存したのか。どこが良かったのか。Markdown でそのまま書ける、小さなノート機能。

03 / Private

すべて、端末の中で完結する

自社サーバーへの送信なし、広告 ID の取得なし、追跡 SDK なし。Face ID でロックも可能。

04 / Share

共有シートから、一瞬で保存

Safari や Instagram、YouTube の共有メニューから VideoClip を選ぶだけで、動画 URL を取り込めます。

05 / Search

タイトルでもメモでも、横断検索

思い出したキーワードひとつで、フォルダを越えて見つけ出せます。

06 / AI (Optional)

必要なときだけ、AI に要約させる

自分で取得した Gemini API キーを設定すれば、動画の要約を端末上で生成可能。API を使わなければ通信は発生しません。

FAQ

よくある質問

動画そのものをアプリ内に保存するのですか？

いいえ。YouTube や Instagram などホスティング元の URL を保存し、再生時にそれらのサービスと直接通信します。動画ファイルを当社サーバーに送ることはありません。

広告や行動追跡はありますか？

ありません。AdMob や Firebase Analytics、行動追跡 SDK は一切使用していません。広告 ID (IDFA) の取得も行いません。

AI 要約機能は必ず使う必要がありますか？

いいえ。ご自身で取得した Google Gemini API キーを設定した場合のみ動作します。未設定なら通信は発生せず、他の機能には影響しません。

課金は解約できますか？

月額・年額サブスクリプションはいつでも iOS の「設定」→「Apple ID」→「サブスクリプション」から解約できます。買い切りプランは解約の概念がありません。

複数端末で同期できますか？

現在は端末内ローカル保存のみです。iCloud 同期は将来のアップデートで検討中です。
