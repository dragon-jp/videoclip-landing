Features

通知も、おすすめも、広告もありません。見返したい動画と、あなたの思考のためだけの場所です。

01 — Organize

フォルダで、意味ごとに分ける

Annotate

FAQ

よくある質問

動画そのものをアプリ内に保存するのですか？

いいえ。YouTube や Instagram などホスティング元の URL を保存し、再生時にそれらのサービスと直接通信します。動画ファイルを当社サーバーに送ることはありません。

広告や行動追跡はありますか？

ありません。AdMob や Firebase Analytics、行動追跡 SDK は一切使用していません。広告 ID (IDFA) の取得も行いません。

AI 要約機能は必ず使う必要がありますか？

いいえ。ご自身で取得した Google Gemini API キーを設定した場合のみ動作します。未設定なら通信は発生せず、他の機能には影響しません。

課金は解約できますか？

月額・年額サブスクリプションはいつでも iOS の「設定」→「Apple ID」→「サブスクリプション」から解約できます。買い切りプランは解約の概念がありません。

複数端末で同期できますか？

現在は端末内ローカル保存のみです。iCloud 同期は将来のアップデートで検討中です。
